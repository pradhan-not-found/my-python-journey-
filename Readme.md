## 📚 Curriculum Overview

This course is divided into five core units, each consisting of approximately 9 lecture hours.

### 🔹 Unit I: Introduction to Python (9 Hours)
Building the foundation of Python programming.
- [ ] **Datatypes & Variables**
- [ ] **Expressions & Statements**
- [ ] **Conditions & Loops**
- [ ] **Functions & Modules**
- [ ] **Data Structures** (Lists, Tuples, Dictionaries, Sets)
- [ ] **Classes & Objects** (OOP basics)
- [ ] **File I/O**
- [ ] **Packages & Environment Management**

### 🔹 Unit II: Data Handling with Pandas (9 Hours)
Mastering data manipulation and analysis using the Pandas library.
- [ ] **Introduction to Series & DataFrames**
- [ ] **Loading & Saving Data** (CSV, Excel, JSON, SQL)
- [ ] **Data Alignment & Missing Data Handling**
- [ ] **Reshaping & Pivoting**
- [ ] **Slicing, Indexing & Subsetting**
- [ ] **Insertion & Deletion**
- [ ] **Merge, Join & Concatenate**
- [ ] **Time Series Analysis**

### 🔹 Unit III: Numerical Computation with NumPy (9 Hours)
High-performance numerical and mathematical operations.
- [ ] **N-dimensional Arrays (ndarrays)**
- [ ] **Array Datatypes & Memory Layout**
- [ ] **Mathematical & Logical Operations**
- [ ] **Linear Algebra operations**
- [ ] **Fourier Transforms**
- [ ] **Random Number Generation**
- [ ] **Searching & Sorting Arrays**
- [ ] **Importing & Exporting Data**

### 🔹 Unit IV: Scientific Computations with SciPy (9 Hours)
Advanced mathematical and scientific computing tools.
- [ ] **Physical & Mathematical Constants**
- [ ] **Integration Routines**
- [ ] **Interpolation Techniques**
- [ ] **Optimization Algorithms**
- [ ] **Signal Processing**
- [ ] **Sparse Matrices**
- [ ] **Spatial Data Structures & Algorithms**
- [ ] **Special Mathematical Functions**
- [ ] **Statistical Functions & Distributions**

### 🔹 Unit V: Data Visualization (9 Hours)
Communicating data insights through compelling visual representations (Matplotlib, Seaborn, Plotly).
- [ ] **Chart Properties & Styling**
- [ ] **Box Plots & Heatmaps**
- [ ] **Scatterplots & Bubble Charts**
- [ ] **3D Charts**
- [ ] **Time Series Visualizations**
- [ ] **Geographical Data Plotting**
- [ ] **Graph/Network Data Visualization**

- **Language**: Python 3.x
- **Data Manipulation**: `pandas`
- **Numerical Computing**: `numpy`
- **Scientific Computing**: `scipy`
- **Visualization**: `matplotlib`, `seaborn`, `plotly`
